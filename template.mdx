### Adding Code Blocks with Syntax Highlighting

Wrap code blocks in triple backticks followed by the language name (e.g. `python`, `javascript`, `css`, `bash`, `json`) to trigger automatic syntax highlighting:

```python
import anthropic

client = anthropic.Anthropic()

# Trigger an execution block
response = client.messages.create(
    model="claude-opus-4-8",
    max_tokens=1000,
    messages=[
        {"role": "user", "content": "Hello, Claude!"}
    ]
)
print(response.content[0].text)
```
